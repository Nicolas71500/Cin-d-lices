# CinéDélices

Salut ! Voici comment installer notre magnifique projet !
Des recettes de cuisines inspirés de films, mangas ou séries !

## Prérequis

Avant de commencer, assure toi d'avoir installé Node.Js (c'était pas le cas pour l'un d'entre nous, mais chut!)

## Installation

Pour installer le projet, suis ces étapes :

1. Clone notre dépôt :
   git clone <git@github.com>:O-clock-Pancakes/cinedelices-back.git

2. Installe les dépendances :

```bash
npm install
npm install express bcrypt cors dotenv ejs zod jest.
```
## Utilisation

Pour démarrer le serveur
```bash
npm run dev
```
